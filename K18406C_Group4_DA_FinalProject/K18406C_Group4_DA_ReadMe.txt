2. How to use:

   2.1. Notes:
	Before you continue, ensure you meet the following requirements:
	* You have installed Microsoft SQL Server 2019 or previous versions
	* You have installed Visual Studio 2019 or previous versions (Need to ensure you have installed SQL Sever Integration Service Project and Microsoft Analysis Services Projects)
	* You have installed Tableau Application and Power BI desktop Application
	* You have installed Microsoft Excel 2019 or previous version which supports PivotTable 
	* You already have known how to use these tools before
	* You have a basic understanding of Database, Data Analysis

   2.2. Running project:
- Step 1: Download file K18406C_Group4_DA_Backup.bak and restore it into SQL Server
- Step 2. Download and extract K18406C_Group4_DA_SSIS.rar and K18406C_Group4_DA_SSAS.rar to run SSIS, SSAS package
- Step 3. Open Excel, Power BI and Tableau files
